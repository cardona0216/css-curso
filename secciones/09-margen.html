Margin - margenes

en esta seccion vimos lo que son los margenes

En CSS, la propiedad margin se utiliza para especificar el espacio alrededor de
un elemento, es decir, la separación entre el borde exterior del elemento y los elementos adyacentes.
Al igual que con el padding, el margin se puede aplicar individualmente a cada lado del elemento o
en forma abreviada para todos los lados a la vez. Aquí tienes ejemplos de ambas formas:

Aplicación individual a cada lado:

/* Aplicar margin individualmente */ margin-top: 10px; margin-right: 20px; margin-bottom: 10px; margin-left: 20px;

Forma abreviada:

/* Aplicar margin en forma abreviada (en el sentido de las agujas del reloj: arriba, derecha, abajo, izquierda) */
margin: 10px 20px 10px 20px; /* O, de manera más corta, especificando solo tres valores: arriba, derecha + izquierda, abajo */
margin: 10px 20px 10px; /* O aún más corta, solo dos valores: arriba + abajo, izquierda + derecha */
margin: 10px 20px; /* O simplemente, un solo valor para aplicar el mismo margin a todos los lados */
margin: 10px;
