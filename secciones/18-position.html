Position

En CSS, la propiedad position se utiliza para definir el método de posicionamiento de un elemento en una página web.
Hay varios valores que puedes utilizar con la propiedad position. Aquí tienes una descripción de los valores más comunes: es decir depende de como queremos interactuar con el elemento sera la posicion
son 5 valores que puede tomar position

position static

static: Este es el valor predeterminado. Los elementos con position: static se colocan en el flujo normal del documento,
lo que significa que se mostrarán en el orden en que aparecen en el HTML.
.ejemplo { position: static; }

position relative

relative: Un elemento con position: relative se posiciona en relación con su posición normal. Es decir, puedes moverlo de su
posición original utilizando las propiedades top, right, bottom y left.
.ejemplo {
position: relative;
top: 10px; left: 20px;
}

posicion fixed

fixed: Un elemento con position: fixed se posiciona con respecto a la ventana del navegador y permanece fijo incluso cuando se desplaza la página. es decir lo pega en algun lado, lo saca de donde estaba y lo podemos pegar en cualquien lado y este se quedara ahi, si hacemos scroll no le prestara
atencion osea pegara el elemento en algun lado
